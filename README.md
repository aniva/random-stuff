# Random Stuff

My personal repository for storing random scripts, snippets, and projects that don't fit into larger, more organized repositories. This is a catch-all for things I find useful, experimental, or just want to keep track of.

## Structure

The repository is organized into folders based on the type of content:

- `scripts/`: Executable scripts and command-line tools.
- `notes/`: Personal notes, snippets, and references.
- `projects/`: Larger experimental projects.
- `configs/`: Configuration files and templates.

## Usage

Simply clone or copy the files you need. Most scripts are written in Python, but you'll find a mix of languages depending on what I was working on at the time.
